# 上线文件导出与后台上传说明

## 需要导出的3个文件

将下列 Markdown 文件分别用 Word、WPS 或 Typora 打开并导出为 PDF：

1. `02_服务商隐私政策_正式版.md` → `智云名片隐私政策.pdf`
2. `03_第三方服务协议_正式版.md` → `智云名片第三方服务协议.pdf`
3. `05_功能及定价表.md` → `智云名片功能及定价表.pdf`

## PDF 导出要求

- A4 纵向，正常页边距。
- 中文字体使用微软雅黑、思源黑体或宋体。
- 保留文件标题、更新/生效日期、运营主体及联系方式。
- 导出后逐页检查，不得出现乱码、内容截断、空白页或 Markdown 符号异常。
- 文件名不使用“草案”“示例”“测试”等字样。
- 不要上传项目内部审核清单或含“提交前填写”的内部测试记录。

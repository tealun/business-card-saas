# 智云名片审核截图拍摄清单

## 建议上传的4张截图

### 图1：员工名片首页

- 企业微信移动端打开智云名片。
- 展示真实测试成员头像、姓名、职务、企业名称和主要操作按钮。
- 不出现加载失败、演示模式、测试提示或空白字段。

### 图2：名片资料编辑

- 展示姓名、职务、联系方式等可编辑区域。
- 使用测试成员本人或已获授权的测试数据。
- 不展示数据库、Token、AppSecret、内部管理地址。

### 图3：访客看到的名片

- 从分享入口打开公开名片。
- 展示企业资料、员工资料和正常可用的联系动作。
- 确认隐私开关关闭的字段没有出现在截图里。

### 图4：企业微信资料授权

- 展示“同步企微资料”入口或企业微信头像、二维码授权页面。
- 不截取真实授权票据、URL 查询参数或后台密钥。

## 图片要求

- 至少上传2张，建议4张。
- 使用竖屏移动端截图，保持清晰、完整、方向正确。
- 截图内容必须与本次审核版本一致。
- 不使用设计稿、拼接营销图、其他应用截图或虚假功能。
- 如截图含手机号或二维码，必须取得对应人员授权；审核测试完成后可更换测试资料。

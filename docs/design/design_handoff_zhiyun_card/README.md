# Handoff：智云名片（ZHIYUN CARD）微信小程序

## Overview
「智云名片」是一款面向 B 端企业的**数字名片**小程序：企业开通后为员工统一下发名片，员工补全资料即可对外分发；访客无需授权即可查看名片落地页并回连、交换。本包覆盖 M1 阶段全部页面、LOGO、名片模板、可换肤色彩体系与非正常态。

## About the Design Files
本包内的 HTML 文件（`智云名片 设计稿.dc.html` + `support.js`）是**设计参考稿**——用 HTML 制作的高保真原型，用于表达最终视觉与交互意图，**不是可直接搬运的生产代码**。
- 目标是在**微信小程序原生框架（WXML/WXSS/JS 或 TS + Skyline）**中，按小程序既有工程规范**重建**这些设计；若团队使用 Taro / uni-app 等跨端框架，则用其组件体系落地。
- `.dc.html` 是一个组件化预览文件，打开后是一块可缩放画布，从上到下分 5 个 turn：
  - **turn 5**：LOGO 导出资源 + 开发 Token 表
  - **turn 4**：状态变体（空/加载/失效/错误）
  - **turn 3**：名片样式、名片夹、企业名片
  - **turn 2**：色彩体系与换肤机制
  - **turn 1**：LOGO 三方案、5 种名片模板、M1 四个核心页
- 直接用浏览器打开该 HTML 即可审阅；`support.js` 为其运行时，需与 HTML 同目录。

## Fidelity
**高保真（hifi）**。颜色、字号、间距、圆角、组件规格均为最终值，请按 Token 表**像素级还原**，用小程序原生组件与项目既有封装实现。图片类资源（头像、企业环境图、荣誉证书、视频封面）在稿中为占位框，需替换为真实素材。

## 设计基准
- 画布宽度 **375px**（iPhone 标准逻辑宽度，小程序用 rpx 时 1px ≈ 2rpx，750rpx 满宽）。
- 字体：系统默认（PingFang SC / 微软雅黑回退），不引入自定义字体。
- 安全区：底部固定操作栏 `padding-bottom: 20–24px`，兼容 Home 指示条（用 `env(safe-area-inset-bottom)`）。

## Screens / Views

### 1. 首页 · 我的名片（tabBar 首页，`data-screen-label="首页-我的名片"`）
- **Purpose**：员工工作台——展示自己的名片、快捷编辑、发名片、看数据与交换请求。
- **Layout**：竖向滚动。状态栏(24) + 导航(44，标题居中，右侧胶囊为小程序原生菜单占位) + 内容(左右边距 16，卡片间距 12) + 底部 tabBar(3 项：首页/名片夹/企业名片)。
- **Components**（自上而下）：
  1. **名片卡** 343×216，白底 r12，左侧 6px 主色条；企业名 + LOGO(20px)、姓名(22/700)、职位副标(14 #6B7280)、右上头像(56 圆形 tint 底)、联系方式(12 #6B7280)、右下状态徽标「正常展示中」(10 #17B26A / #E7F7EF r6)。
  2. **编辑快捷栏** 白卡三等分：编辑资料 / 名片样式 / 切换名片（图标 18 + 文字 13，竖分隔线 #EDEFF2）。
  3. **主按钮「发名片」** H48 主色 r10，含纸飞机图标，阴影 `0 4px 12px rgba(43,108,255,.28)`。
  4. **工具入口** 白卡 5 列宫格：名片图片 / 文字转名片 / 名片榜单 / 名片码 / 全部（图标 24 主色描边 + 文字 12）。
  5. **交换请求** 白卡：标题 + 「有 1 人请求…」；一行请求项（头像44 + 姓名职位 + 忽略[描边]/同意[主色] 按钮 H32）。
  6. **我的名片数据** 白卡：3 列统计（访客328 主色 / 我看过56 / 好友新名片4），下方最近访客列表（头像40 + 姓名+职位 + 访问次数/交换状态 + 时间）。
- **tabBar**：首页选中（主色实心图标 + 主色文字），其余灰 #9AA1AC。

### 2. 发名片（底部弹层，`data-screen-label="发名片弹层"`）
- **Purpose**：选择分发方式。
- **Layout**：半透明遮罩 `rgba(26,29,31,.45)` + 底部 sheet（白，顶部 r16，抓手条 36×4 #EDEFF2）。
- **Components**：标题栏(发名片 + 关闭×) → 当前名片缩略条(灰底 r12) → **主推 2 格**(tint 底 r12：转发给好友/群、生成名片海报) → **次级 2 格**(描边 r12：名片小程序码、复制链接) → **海报预览**(左 88×140 海报缩略 + 右说明 + 保存到相册按钮) → 隐私说明文字。

### 3. 编辑资料（`data-screen-label="编辑资料"`）
- **Purpose**：补全/修改名片信息与隐私开关。
- **Layout**：导航含返回箭头 + 标题；分组白卡；底部固定「保存并更新名片」H48 主色。
- **Components**：
  - **基础信息** 卡：头像(48 圆) / 姓名 / 职位 / **部门**(灰+锁图标「企业统一维护」不可改) / 个人简介。行高约 44–48，右侧值 + `›`，分隔线 #EDEFF2。
  - **联系方式** 卡：手机 / 座机 / 邮箱 / 微信号（可改）+ 公司地址 / 官网（锁，企业维护）。
  - **展示标签** 卡：主色 tint 芯片 + 虚线「+ 添加标签」。
  - **隐私设置** 卡：4 个开关（手机/邮箱/微信号对访客公开、展示企业微信入口）。开关 44×24，开=主色。
  - 底部灰字说明：锁字段由企业维护、敏感字段默认不公开。

### 4. 访客名片详情（分享落地页，`data-screen-label="访客名片详情"`）
- **Purpose**：外部访客查看名片、回连、交换。**无需授权**。
- **Layout**：主色渐变头图 hero（160°，brand-deep→brand，含 1px 点阵纹理）+ 白底内容区 + 底部固定操作栏。
- **Components**：
  - **Hero**：导航(返回+姓名居中，白色) → 企业名+LOGO → 姓名(26/700 白) + 职位 + 头像(64 圆 白底) + 联系方式(13 白 85%)。
  - **互动区** 白卡：328 人看过 / 靠谱 26 + 右侧「在职认证」「官方认证」徽标。
  - **联系动作宫格** 4 列×2 行：电话/邮箱/地址/微信/纸质名片/存通讯录/转发名片/加企微（图标 24 主色 + 文字 12）。
  - **公司介绍** 卡：正文 + 2 图占位(办公环境/团队合照，96 高虚线框) + 展开全文。
  - **企业视频** 卡：176 高深色播放器占位（播放按钮 52 圆 + 时长角标）。
  - **公司荣誉** 卡：横向滑动卡片(180 宽，证书图占位 120 高 + 标题/副标)，底部指示点。
  - **底部固定操作栏**：加企微(48 图标钮) + 转发(48 图标钮) + 保存电话(描边) + 拨打电话(主色实心)。

### 5. 名片样式（`data-screen-label="名片样式"`）
- **Purpose**：所见即所得地选模板 + 品牌色 + 名片码风格。
- **Components**：顶部**实时预览**名片(右下「实时预览」徽标) → **选择模板**卡(横向缩略，选中 2px 主色边+勾) → **品牌主色**卡(6 圆色 + 自定义，选中双环) → **名片码样式**卡(进入行) → 底部「应用此样式」H48 主色。

### 6. 名片夹（tabBar，`data-screen-label="名片夹"`）
- **Purpose**：员工侧访客/联系人管理。
- **Components**：顶部 tab（我的访客328 / 我看过的56 / 好友名片4，选中主色下划线）→ 搜索框(H38 + 筛选) → 按「今天/本周」分组的白卡列表：头像44(按状态配色) + 姓名职位 + 公司/访问次数 + 右侧交换状态徽标(已交换绿 / 待通过橙 / 未交换灰 / 「交换」描边钮) + 时间。

### 7. 企业名片（tabBar，`data-screen-label="企业名片"`）
- **Purpose**：全公司统一对外主页。
- **Components**：主色渐变 hero（企业 LOGO 64 白圆角 + 企业名 + 已认证徽标 + 数据条：在职成员/服务客户/成立年份）→ **企业简介**卡(正文 + 资质芯片) → **联系与官网**卡(地址/电话/官网行) → **团队成员**卡(按部门分组，头像40 + 姓名[本人标] + 职位 + 「名片」入口) → 底部「进入官网[描边] / 关注企业[主色]」。

## 状态变体（turn 4）
- **首页空态**：新员工未建名片。120 圆角方 tint 底大 LOGO + 「还没有你的数字名片」+ 说明 + 「创建我的名片」主按钮 + 「从企业通讯录导入」文字链。
- **名片夹空态**：100 圆灰图标 + 「还没有访客记录」+ 「去发名片」按钮。
- **加载骨架屏**：列表灰块占位（头像圆 + 两行条），底部「加载中…」。块色 `#EDEFF2` / `#F0F2F5`，实现时加轻微 shimmer。
- **名片失效态**（离职/停用）：hero 转**灰**渐变(#5A6270→#7A828F)，职位前缀「原」；橙色警示条「该名片已停用」(#FEF3E5 底 / #FBD9A5 边 / #B54708 字)；旧联系方式置灰 opacity .55 标「已失效」；CTA 改为「查看企业名片」。
- **加载失败**：断网图标(红斜杠) + 「加载失败」+ 说明 + 「重新加载」描边按钮。

## Interactions & Behavior
- 发名片：首页主按钮 → 底部弹层升起（遮罩淡入，sheet transform 上滑，~250ms ease-out）。
- 编辑资料：锁定字段点击无响应（或 toast「该字段由企业统一维护」）；保存后名片实时更新并返回。
- 访客详情：拨打电话调 `makePhoneCall`；存通讯录调 `addPhoneContact`；转发走 `button open-type="share"`；加企微跳企业微信。
- 名片夹：tab 切换列表；「交换」按钮 → 待通过状态；搜索/筛选。
- 交换请求：同意/忽略即时更新首页列表与徽标计数。
- 状态切换：加载中→成功渲染 / 失败→错误态（可重试）；名片被停用→失效态。

## State Management
- 当前用户名片（`myCard`）、名片模板与品牌色（`themeConfig`：`brand` 主色，其余三档由主色推导）。
- 访客/联系人列表（分组、交换状态、分页）、交换请求队列与未读计数。
- 名片数据统计（访客数/浏览/靠谱数）。
- 页面级 UI 状态机：`loading | ready | empty | error | disabled`。
- 数据获取：名片详情按公开编号请求（分享链接**仅含公开编号，不含内部字段**）；隐私开关决定敏感字段是否下发。

## Design Tokens
**品牌（可换肤，企业开通时选定，仅此一档随企业变化）**
- `--brand` #2B6CFF（默认智云蓝） · `--brand-deep` #1E56D6 · `--brand-tint` #EAF1FF · `--brand-soft` #A9C4FF
- 推导规则：deep = 主色亮度 ×0.8；tint = 主色 8% 铺白；soft = 主色 40% 混白。
- 预设可选：商务红 #E5484D / 科技紫 #7C5CFC / 沉稳墨绿 #1F8A5B / 活力橙 #F5820D / 深海青 #0E9C9C（各自推导 deep/tint）。

**中性 / 背景（固定）**
- `--text-primary` #1A1D1F · `--text-secondary` #6B7280 · `--text-tertiary` #9AA1AC
- `--bg-page` #F4F6F8 · `--bg-surface` #FFFFFF · `--divider` #EDEFF2

**功能（固定语义）**
- `--success` #17B26A（在职认证/已交换）· `--warning` #F79009（待处理/停用提示）· `--danger` #F04438（错误）
- 功能色 tint：成功 #E7F7EF、警告 #FEF3E5。

**字号 / 字重**：26/700 大标题 · 22/700 名片姓名 · 18/500 导航与卡片标题 · 16/500 区块标题与按钮 · 14/400 正文 · 12/400 辅助。

**圆角**：徽标/芯片 6 · 按钮 10 · 卡片 12 · 弹层/大图标块 16 · 头像 50%。

**间距（8pt 栅格）**：4 图标↔文字 · 8 元素间 · 12 卡片间距 · 16 页面左右边距/卡内边距 · 20 名片卡内边距。

**阴影**：卡片 `0 2px 8px rgba(16,24,40,.06)`；主按钮 `0 4px 12px rgba(43,108,255,.28)`（换肤时随 brand）。

**组件规格**：主/次按钮 H48 r10（次按钮 1.5px 描边）；开关 44×24（开=brand）；列表头像 40/44；名片头像 56/64；名片卡 343×216（1.586:1，主色条 6px）；图标线宽 1.5px，尺寸 20/24；最小点击区 44px；访客 hero 渐变 160° brand-deep→brand。

## Assets（`assets/logo/`）
LOGO 定稿为「云 + 名片卡」方案，已导出：
- **主标横版**：`lockup-color-white-1200.png`（白底）、`lockup-white-transparent-1200.png`（反白，深底用）、`lockup-color-transparent-1200.png`。
- **小程序图标（蓝底）**：`icon-blue-{1024,512,256,144,108}.png`。
- **图标变体**：`icon-white-{512,144}.png`（白底）、`mark-color-{512,144}.png`（透明彩色）、`mark-mono-black-512.png`（墨稿单色）、`mark-mono-white-512.png`（反白）。
- 最小使用：图标 ≥24px，主标 ≥88px 宽；四周留白 ≥ 图标高度 20%。
- 页面中的头像、企业环境图、荣誉证书、视频封面均为**占位**，需替换真实素材。图标建议按 Token 用小程序矢量/字体图标或 SVG 还原（线宽 1.5px）。

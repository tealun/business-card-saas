2色彩体系 — 固定基色 + 企业可换的品牌主色

机制：整套 UI 只有一个随企业变化的变量——品牌主色（--brand）。其余中性色、功能色、背景全部固定不变，保证任何品牌色下都清晰易读。企业开通时从预设色中选一个，或输入自有品牌色 HEX；系统自动推导出「深色 / 浅底 / 描边」三档，套用到名片卡、按钮、访客页头图、头像底、标签等所有主色出现的位置。
